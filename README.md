# Expense Tracker

## Web app designed to keep track and maintain a clear history of your daily expenses.


![App Screenshot](docs/Front_Page.png)

## Tech Stack: **MERN**
- MongodbAtlas;
- Express.js; 
- React.js;
- Node.js;

## Dependencies (beyond the stack):
- Cors;
- Dotenv;
- Mongoose;
- TailwindCSS;
- Lucide-react;

## How to run locally

1. Clone the repository
2. Create a `.env` file in the root with your MongoDB connection string:
```
   MONGODB_URL=your_connection_string
```
3. Install dependencies:
```
   cd server && npm install
   cd client && npm install
```
4. Run the server: `node server/server.js`
5. Run the client: `npm run dev`

## Features:
- Add expenses;
- View expenses;
- Update expenses;
- Delete expenses;
- Filter expenses by category;
